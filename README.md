# Tic Tac Toe Game 🎮

A simple Tic Tac Toe game made using HTML, CSS, and JavaScript.

## Link of Game 🎮


## Features ✨
- Two player game (O vs X)
- Winner detection
- Reset button
- Simple and clean UI
- Interactive gameplay

## Technologies Used 🚀
- HTML
- CSS
- JavaScript

## How to Play 🎯
1. Player O starts first.
2. Players take turns clicking boxes.
3. First player to match 3 symbols in a row wins.
4. Click the Reset button to start a new game.

## Winning Conditions 🏆
The game checks these winning patterns:

- Horizontal
- Vertical
- Diagonal
